# Sample-scaffolding

这是一个使用 node.js 编写的小型简易脚手架工具

## 开始

这些说明将为您提供一个在你的个人本地计算机上启动和运行的项目副本,以便进行开发和测试.

## 环境

- 操作系统: Windows10/MacOS
- 编辑器: Visual Studio Code
